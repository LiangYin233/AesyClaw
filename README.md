**AesyClaw** 是一个可扩展的 AI Agent 运行时平台。

## 快速开始

```bash
# 安装依赖
yarn install

# 启动开发服务器
yarn dev

# 启动（生产模式）
yarn start

# 构建 Web UI
yarn build

# 运行测试
yarn test
```

## 技术栈

| 类别       | 技术                       | 说明                 |
| ---------- | -------------------------- | -------------------- |
| Web 框架   | Hono 4                     | 轻量高性能 HTTP 服务 |
| Agent 引擎 | pi-agent-core / pi-ai 0.70 | Agent 核心与 AI 抽象 |
| 协议集成   | MCP SDK                    | 模型上下文协议       |
